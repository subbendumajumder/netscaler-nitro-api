#lsnip6profile

Configuration for LSN ip6 Profile resource.


##Properties 
<span>(click to see [Operations](#opera))</span>


<table><thead><tr><th>Name</th><th>Data Type</th><th>Permissions</th><th>Description</th></tr></thead><tbody><tr><td>name</td><td>&lt;String></td><td>Read-write</td><td>Name for the LSN ip6 profile. Must begin with an ASCII alphanumeric or underscore (_) character, and must contain only ASCII alphanumeric, underscore, hash (#), period (.), space, colon (:), at (@), equals (=), and hyphen (-) characters. Cannot be changed after the LSN ip6 profile is created. The following requirement applies only to the NetScaler CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, "lsn ip6 profile1" or 'lsn ip6 profile1').<br>Minimum length = 1<br>Maximum length = 127</td></tr><tr><td>type</td><td>&lt;String></td><td>Read-write</td><td>IPv6 translation type for which to set the LSN IP6 profile parameters.<br>Possible values = DS-Lite, NAT64</td></tr><tr><td>natprefix</td><td>&lt;String></td><td>Read-write</td><td>IPv6 address(es) of the LSN subscriber(s) or subscriber network(s) on whose traffic you want the NetScaler ADC to perform Large Scale NAT.</td></tr><tr><td>network6</td><td>&lt;String></td><td>Read-write</td><td>IPv6 address of the NetScaler ADC AFTR device.</td></tr><tr><td>__count</td><td>&lt;Double></td><td>Read-only</td><td>count parameter</td></tr></tbody></table>
